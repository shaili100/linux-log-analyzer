# Linux Log Analyzer & System Monitor

## 📌 Overview
This project analyzes system logs and monitors system performance using Linux commands and shell scripting.

## 🛠 Tools Used
- Linux
- Bash Scripting
- grep, awk, sed

## 🚀 Features
- Log analysis (errors, warnings)
- System monitoring (CPU, memory, disk)
- Automated report generation

## 📂 Files
- monitor.sh - automation script
- syslog.txt - sample log file

## 🎯 Outcome
Improved troubleshooting and automation skills using Linux.
