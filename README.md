# linux-log-analyzer
# 🚀 Linux Log Analyzer & System Monitor

## 📌 Project Overview
This project is a Linux-based shell scripting tool designed to analyze system logs and monitor system performance. It helps in identifying errors, warnings, and system resource usage.

---

## 🎯 Objective
- To practice Linux commands and shell scripting  
- To analyze log files and extract meaningful insights  
- To simulate real-time system monitoring  

---

## 🛠️ Tools & Technologies
- Linux Commands  
- Bash Scripting  
- Git & GitHub  

---

## 📂 Project Structure

linux-log-analyzer/
│── monitor.sh # Main script for system monitoring
│── syslog.txt # Sample log file
│── README.md # Project documentation


---

## ⚙️ Features
- 📊 System monitoring (Disk & Memory usage)
- 🔍 Log analysis using `grep`
- ⚡ Automated report generation
- 🧪 Simulated real-time log updates

---

## ▶️ How to Run

1. Open terminal (Git Bash)
2. Navigate to project folder:

cd linux-log-analyzer


3. Run the script:

bash monitor.sh
